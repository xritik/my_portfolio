# Ritik Singh — Personal Portfolio

> A modern, responsive personal portfolio website built with pure HTML, CSS, and JavaScript — featuring dark/light mode, animated UI, working contact form, and CV download.

👉 **[https://YOUR_HOSTED_URL_HERE](https://YOUR_HOSTED_URL_HERE)**

---

## 📸 Features

- 🌗 **Dark / Light Mode** — toggle with preference saved to localStorage
- ✍️ **Typed Text Animation** — cycles through roles on the home screen
- 📬 **Working Contact Form** — powered by EmailJS, messages land in your inbox
- 📄 **CV Download** — one-click download of latest resume PDF
- 📱 **Fully Responsive** — mobile, tablet, and desktop ready
- ✨ **Scroll Animations** — smooth reveal effects using IntersectionObserver
- ⚡ **Zero Frameworks** — pure HTML, CSS, JavaScript (no React, no build step)
